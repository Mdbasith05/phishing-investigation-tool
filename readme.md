# 🔍 Phishing Investigation Tool

A cybersecurity tool that instantly analyzes suspicious 
IPs, domains, and URLs against VirusTotal threat intelligence.

## Features
- Automatic IOC type detection (IP/Domain/URL)
- Real-time VirusTotal threat intelligence
- Clean/Suspicious/Malicious verdict
- Country and reputation data for IPs
- Plain English explanation of results

## How To Use
1. Paste any suspicious IP, domain, or URL
2. Click Investigate
3. Get instant threat verdict

## Built With
Python | Streamlit | VirusTotal API
